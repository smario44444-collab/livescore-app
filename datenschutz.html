Datenschutzerklärung

Diese Datenschutzerklärung klärt Sie über die Art, den Umfang und Zweck der Verarbeitung von personenbezogenen Daten innerhalb unserer App "SMARIO444 & FCNILS Live Scores" auf.

1. Datenerfassung

Unsere App ist eine Progressive Web App (PWA). Wir erheben, speichern oder verarbeiten aktiv keine persönlichen Nutzerdaten (wie Namen, E-Mail-Adressen oder Telefonnummern), da keine Registrierung erforderlich ist.

2. Externe APIs und Dienste

Die App ruft Live-Sportdaten von der OpenLigaDB ab. Dabei wird Ihre IP-Adresse aus technischen Gründen an die Server dieser API übermittelt, damit die Daten auf Ihr Gerät gesendet werden können.

3. Push-Benachrichtigungen

Wenn Sie Push-Benachrichtigungen für Tore aktivieren, nutzt die App die Standard-Web-Push-Funktionen Ihres Geräts. Es werden keine gerätespezifischen IDs oder Profile auf unseren Servern gespeichert. Die Einstellungen werden lediglich lokal auf Ihrem Gerät (im Browser-Speicher) abgelegt.

4. Werbung & Links

Diese App kann Werbung (z.B. Affiliate-Links) enthalten. Beim Klick auf diese Links verlassen Sie unsere App und es gelten die Datenschutzbestimmungen der jeweiligen Drittanbieter (z.B. Amazon, Fan-Shops).

5. Kontakt

Bei Fragen zum Datenschutz rund um diese App können Sie uns über unsere YouTube-Kanäle kontaktieren.
